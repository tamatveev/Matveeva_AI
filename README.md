# Matveeva AI

Telegram-бот на базе LLM: знакомит потенциальных клиентов с услугами, расценками и примерами работ, помогает оформить заявку. Диалог строится через кнопки (готовые варианты ответов), без обязательного ввода текста. Данные об услугах и примерах подгружаются с Google Drive.

**Автор:** Тимофей Матвеев  

**Статус:** Начальная стадия (идея и планирование)
